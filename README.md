# workshop-graphql

:ferris_wheel: GraphQL workshop practice at MedJS - 01-01-2018

## Build Setup

``` bash
# clone repository
git clone https://github.com/lexmartinez/workshop-graphql.git

# open project folder
cd workshop-graphql

# install dependencies
npm install

# run app
npm start
```
